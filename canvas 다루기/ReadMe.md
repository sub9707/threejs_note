## 설치

**개발환경은 react-vite**

```bash
npm install
```

## 출처

fastcampus - 21개 프로젝트로 완성하는 인터랙티브 웹 개발 with Three.js & Canvas. <br/>
html/css로 제작된 강의 내용을 react 용으로 옮겨 담아 리액트 라이프 사이클에 맞게 수정하였음.<br/>
